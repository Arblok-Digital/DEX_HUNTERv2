**DEX HUNTER** adalah terminal trading *high-performance* yang dirancang untuk berburu token *early-stage* dan *low-cap* di ekosistem Solana, Ethereum, dan Base. Menggunakan algoritma **Institutional Brain** untuk mendeteksi pergerakan *Smart Money* sebelum terjadi lonjakan harga.

## ⚡ Fitur Utama

- **🐋 Global Whale Tracker:** Pantau aktivitas akumulasi paus secara real-time dengan filter volume transaksional >$5K.
- **🧠 Institutional Brain:** Analisa struktur pasar (HH/LL), CVD Delta, dan efisiensi mikro-velocity untuk membedakan akumulasi asli vs *wash trading*.
- **💊 Pump.fun Insider Tracker:** Deteksi dini token baru di Pump.fun dengan pemindaian otomatis terhadap holder dan top 10 ratio.
- **💸 Money Flow Engine:** Skor "Hunter" otomatis (0-100) berdasarkan konfluensi likuiditas, volume, dan risiko kontrak.
- **⚡ One-Click Swap:** Integrasi Jupiter Swap yang mendukung referral sistem untuk eksekusi trade super cepat.
- **📊 Professional Reporting:** Laporan analisa harian otomatis untuk token-token potensial (High Conviction).

## 🛠️ Tech Stack

- **Frontend:** Vanilla JS, CSS3 (Wall Street Ticker UI), HTML5.
- **Backend:** Node.js, Express, Vite.
- **Database:** Firebase Firestore (Centralized Config).
- **APIs:** DexScreener, Jupiter, PumpPortal, Birdeye.

## 🚀 Quick Start

1. **Clone Repo**
2. **Install Dependencies:**
   ```bash
   npm install
   ```
3. **Setup Environment:**
   Copy `.env.example` ke `.env.local` dan masukkan `GEMINI_API_KEY` serta `COINMARKETCAP_API_KEY` lu.
4. **Run Dev:**
   ```bash
   npm run dev
   ```
